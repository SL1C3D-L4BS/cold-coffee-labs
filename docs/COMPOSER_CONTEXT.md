## Non-Negotiables — The Rules You Cannot Break

| Rule | What it means |
|------|--------------|
| **C++23 only for gameplay** | No scripting language. No VM. No Lua. No Python in binaries. Logic = compiled C++23 code in `gameplay/`. Offline Python in `tools/`: `docs/01_CONSTITUTION_AND_PROGRAM.md` §2.2.1. |
| **Clang only** | `clang-cl` on Windows, `clang` on Linux. MSVC and GCC: never. |
| **Rust only in `bld/`** | BLD AI copilot only. Nowhere else. |
| **POD ECS components** | All ECS components are plain data structs. No virtual functions in components. |
| **Pure systems** | All ECS systems are pure functions over component spans. No global state reads in tick. |
| **144 FPS on RX 580** | Every Tier A feature hits 144 FPS at 1080p on an AMD Radeon RX 580 8GB. |
| **Arena-scale only** | No planetary simulation, no orbital mechanics, no galaxy generation. Sacrilege is nine arenas. |
| **No raw new/delete** | Use `std::unique_ptr`, `gw::Ref<T>`, stack scope, or arena allocators. |
| **No `[=]` or `[&]` lambdas** | Explicit captures only. |
| **doctest for everything** | Every new system has doctest coverage. CI must be green. |

## What *Sacrilege* Is

- **Nine Inverted Circles of Hell** — each procedurally generated from a Hell Seed
- **Same seed = same level always** — deterministic, speedrun-shareable
- **Martyrdom Engine** — Sin (0–100) accrues from Profane weapons; at 100 → Rapture (6s god-mode) → Ruin (12s debuff); spend Sin on Blasphemies to manage the meter
- **Speed-first movement** — bunny hop, slide, wall kick, grapple, blood surf; custom swept-AABB controller
- **6 weapons** — Halo-Scythe, Dogma×2, Heretic's Lament, Confessor, Schism, Cathedral; no reloads; all have alt-fires
- **8 enemy types** — Cherubim, Deacon, Leviathan, Warden, Martyr, Hell Knight, Painweaver, Abyssal
- **Persistent gore** — severed limbs are nav obstacles; blood decals enable blood surf
- **God Machine** — 4-phase final boss in a collapsing cathedral

## ECS Pattern Reference

```cpp
// CORRECT: POD component
struct SinComponent { float value = 0.f; float max = 100.f; float accrual_rate = 5.f; };

// CORRECT: Pure system
void SinAccrualSystem::tick(gw::ecs::World& world, float dt) {
    // Use explicit lambda captures in real code (`docs/01` §3) — e.g. [dt] not [&].
    world.query<SinComponent, const ProfaneHitTag>([dt](SinComponent& sin, const ProfaneHitTag&) {
        sin.value = std::min(sin.value + sin.accrual_rate * dt, sin.max);
    });
}

// WRONG: virtual in component (never do this)
struct SinComponent { virtual void tick() {} };  // FORBIDDEN

// WRONG: global state in system
void SinAccrualSystem::tick(...) {
    g_global_sin_value += dt;  // FORBIDDEN — pure function only
}
```
